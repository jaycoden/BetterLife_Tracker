## Important: First Launch Security

The first time you double-click "Life OS.app", macOS might say:

> "Life OS.app cannot be opened because it is from an unidentified developer"

**To fix this:**

1. Right-click (or Control-click) on "Life OS.app"
2. Select "Open"
3. Click "Open" in the dialog
4. This only needs to be done once!

Do the same for "Stop Life OS.app" if needed.

- Make sure `.env.local` is configured
- Check `lifeos.log` file in the project folder

- The app will detect it and just open your browser
